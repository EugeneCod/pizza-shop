npm i react-redux

Фейковый сервер для тестирования запросов из фронта json-server
Команда для запуска при наличии соответствующего файла в дир. public
npx json-server --watch public/db.json

Кто использует npm и имеет проблемы с запуском приложения вместе c json-server, делайте следующее:
1) в консоли npm i -D concurrently;
2) в package.json внутри "scripts" внесите "dev": "concurrently \"npm start\" \"npm run server\"";
3) в консоли npm run dev.
